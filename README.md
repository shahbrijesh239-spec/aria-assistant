# 🌙 Aria — Your Personal AI Assistant

## Setup (one time only)

1. Make sure you have **Node.js** installed → https://nodejs.org
2. Open a terminal in this folder
3. Run: `npm install`
4. Run: `npm start`

## Using on your phone

1. Start Aria with `npm start`
2. The terminal will show something like:
   ```
   On your PHONE : http://192.168.1.5:3000
   ```
3. Open that address in **Safari or Chrome** on your phone
4. **iPhone**: tap Share → Add to Home Screen → it opens like an app
5. **Android**: tap the 3-dot menu → Add to Home Screen

## You need
- An Anthropic API key from **console.anthropic.com** (free signup)
- Your phone and computer on the **same WiFi network**

## Cost
~$1–2 per month at personal use levels.

## Features
- 🗣 Voice input (tap mic) + voice output (Aria talks back)
- 🧠 Memory system — Aria remembers you across conversations
- 📋 Daily check-in — log your day, she responds and saves it
- 😊 Mood detection — tell her how you're feeling
- ✨ Emotion-aware responses
- 4 personality modes: Warm / Motivating / Calm / Playful
